# India-CVD Risk Simulation & Triage Score

This repository contains the full computational framework for the manuscript: 
**"Imported ASCVD Risk Calculators Generate Substantial Treatment Discordance in Indian Primary Prevention"**

## 📂 Repository Contents
- `FINAL_MANUSCRIPT_DATA.csv`: The locked 100,000-patient synthetic dataset anchored to ICMR-INDIAB-17 metrics.
- `ascvd8best.ipynb`: Interactive Jupyter/Colab Notebook containing the simulation logic and figure generation.
- `ascvd8best.py`: Executable Python script for the full analytical pipeline.

## 🚀 How to Reproduce
1. Clone this repository.
2. Ensure you have the dependencies installed: `pip install -r requirements.txt`.
3. Run the analysis: `python ascvd8best.py`.
